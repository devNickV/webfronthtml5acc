Javascript, basic constructs

The following constructs are basic language constructs of javascript:

  • simple types: numbers, strings, booleans (true and false), null, and undefined.
  • variable
    • global
  • operators
  • iterators
  • statements
  • function
  • object
  • this
  • seperator
